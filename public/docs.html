समन्वय | Central Schema Mapper API

The Central Schema Mapper accepts an Amazon ASIN, retrieves product information using an internal Amazon service, transforms the data into a standardized schema, and automatically inserts the normalized document into MongoDB.

Endpoint

GET /v1/map/:asin

Maps an Amazon ASIN into the centralized product schema and stores it in the database.

Request

Parameter Type Required Description
asin String Yes Amazon Standard Identification Number
GET /v1/map/B0FQFPHB1W
